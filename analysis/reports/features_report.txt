--- FEATURES UTILIZADOS (46) ---
Estos features forman el vector por escenario y se usan para correlación Pearson/Spearman,
distancia coseno, clustering y figuras.

  world_area
    Descripción: Área del mundo Wx×Wy (m²)
    Origen:      MovementModel.worldSize

  aspect_ratio
    Descripción: Relación de aspecto min(Wx,Wy)/max(Wx,Wy) ∈ (0,1]
    Origen:      MovementModel.worldSize

  N
    Descripción: Número de hosts
    Origen:      Scenario.nrofHostGroups, Group*.nrofHosts

  density
    Descripción: Densidad proxy (hosts/km²)
    Origen:      N, world_area (derivado); excluida del core por redundancia

  speed_mean
    Descripción: Velocidad media (m/s)
    Origen:      Group*.speed

  pause_ratio
    Descripción: Ratio pausa/(movimiento+pausa)
    Origen:      Group*.waitTime (derivado)

  wait_mean
    Descripción: Tiempo de espera medio (s)
    Origen:      Group*.waitTime

  mm_WDM
    Descripción: Usa WorkingDayMovement (0/1)
    Origen:      Group*.movementModel

  mm_RWP
    Descripción: Usa RandomWaypoint (0/1)
    Origen:      Group*.movementModel

  mm_MapRoute
    Descripción: Usa MapRouteMovement (0/1)
    Origen:      Group*.movementModel

  mm_Cluster
    Descripción: Usa ClusterMovement (0/1)

  mm_Bus

  mm_ShortestPath

  mm_External

  mm_Linear
    Descripción: Usa LinearMovement (0/1)

  transmitRange
    Descripción: Rango de transmisión (m)
    Origen:      bt0.transmitRange / interface.transmitRange

  contact_rate_proxy
    Descripción: Proxy tasa de contacto
    Origen:      density, transmitRange, speed (derivado)

  event_interval_mean
    Descripción: Intervalo medio entre mensajes (s)

  event_size_mean

    Origen:      Events1.size

  msgTtl

  pattern_uniform

  nrof_event_generators

  bufferSize

  transmitSpeed

  workDayLength

    Origen:      Group*.workDayLength

  timeDiffSTD
    Descripción: Desv. estándar diferencia horaria (s); NaN si no WDM
    Origen:      Group*.timeDiffSTD

  probGoShoppingAfterWork

  nrOfMeetingSpots
